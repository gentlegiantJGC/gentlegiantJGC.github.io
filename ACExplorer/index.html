ACExplorer

ACExplorer is a project I have been working on since early 2016. The aim is to extract models and textures from the video game Assassin's Creed Unity with the ultimate aim being to extract whole sections of the world with the click of a button. If this is ever achieved I would like to extend this to the other games in the Assassin's Creed franchise but I want to perfect a single game first.

This project is based on another program called ARchive_neXt however that program is not open source or easily modifiable and at the time of writing the author has not been active for a while. This program should be easy for the community to add to and improve if someone wishes to.
