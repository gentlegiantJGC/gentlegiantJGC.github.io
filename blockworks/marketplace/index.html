Marketplace Projects

I have had a hand in a number of projects that have appeared on the Minecraft Marketplace released by Blockworks. This usually included creating the world in Minecraft Bedrock edition and converting to the Java edition format so that the builders could easily work on the world using server tools. Once they were done with the world it needed converting back to the Bedrock format for the mechanics team to add redstone and other mechanics.

After all this was done the worlds needed packaging up correctly to a format defined by Microsoft and putting through a validtion program to confirm all the files were correct and valid.
